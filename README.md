# Dotfiles

Personal configuration files for my Fedora Linux setup, version-controlled so I can track changes, roll back when needed, and bootstrap a new machine quickly.

## Stack

| Category             | Application | 
| -------------------- | ----------- | 
| Terminal             | Ghostty     |             
| Shell                | Zsh         |              
| Terminal Multiplexer | Tmux        |      
| Top Bar              | Waybar      |        
| Window Manager       | Wayland     | 
| Font                 | Terminus    |
| IDE                  | Zed         | 


## Repository Structure

```
dotfiles/
├── ghostty/          # Ghostty terminal config
├── zsh/              # Zsh shell config (.zshrc, .zprofile, etc.)
├── tmux/             # Tmux config
├── waybar/           # Waybar top bar config & styles
├── wayland/          # Wayland compositor config
├── fonts/            # Terminus font files
└── zed/              # Zed IDE settings
```
